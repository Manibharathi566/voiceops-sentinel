# VoiceOps Sentinel

## Overview
VoiceOps Sentinel is a Real-Time Call Intelligence System that converts customer call audio into text and generates business insights.

## Features
- Speech-to-Text (Whisper)
- Call Summarization
- Sentiment Analysis
- Action Item Extraction
- Speaker Diarization
- PII Redaction

## Tech Stack
- Python
- Streamlit
- OpenAI Whisper
- Pyannote
- SpaCy
